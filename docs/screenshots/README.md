# Screenshot Placement Guide

Add the screenshot image files from your local machine into this folder with these exact names:

- `admin-workspace-hero.png`
- `admin-dashboard-metrics.png`
- `seller-workspace-hero.png`
- `seller-dashboard-form.png`
- `buyer-workspace-hero.png`
- `buyer-catalog.png`

Once these files are present, the main project `README.md` will render them automatically on GitHub.
